Lattitude

Summary: Lattitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

Afther assembling the dataset we used Matplotlib to plot various aspects of the weather. Factors we looked at included temperature, cloudness, wind speed and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanation and description of any trends and correlations witnessed.
